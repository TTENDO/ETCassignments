  1. Element Selector : It matches one or more elements with the same name
  2. ID and Class selector : ID selector matches any HTML element that has an ID Attribute with the same value as that of the selector.
    Class selector matches all elements on the page with the same value of the class attribute
  3. Descedant selector : This selector lets you combine two or more selectors so as to be specific in your selection method.
  4. Direct descendant sector. this is like the descendant selector only that it targets the immediate child elements
  5. Attribute selector. This targets elements based on the presence and value of HTML attribuutes and is declared using square brackets
