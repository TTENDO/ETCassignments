Margin and Padding

  • With every increase in margin (margin-left and margin-right), there is a decrease in the width of an element and with every decrease in the margin-left and margin-right, there is an increase in the width of an element.
  • With every increase in the padding of an element (padding-top an d padding-bottom), there is an increase in the height of an element and with every decrease in the padding of an element, there is a decrease in the height of an element.
  • This kind of behavior can be changed by fixing the width and height of an element using a specific value.
